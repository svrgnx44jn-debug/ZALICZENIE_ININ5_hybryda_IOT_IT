# Zaliczenie przedmiotu - zarządzanie projektem wdrożeniowym

## Opis projektu
Projekt „Przewodnik po stolicach” ma na celu prezentację wybranych stolic świata
w formie prostej strony internetowej. Każda stolica opisana jest w osobnej sekcji
zawierającej nazwę miasta, zdjęcie oraz krótki opis.

W ramach tego repozytorium dodano opis stolicy Tajlandii – **Bangkoku**.
Miasto zostało przedstawione poprzez sekcję HTML zawierającą podstawowe informacje
oraz grafikę umieszczoną w osobnym katalogu.

Projekt jest realizowany w ramach zaliczenia przedmiotu i służy nauce pracy
zespołowej z wykorzystaniem systemu kontroli wersji Git oraz platformy GitHub.

## Zawartość projektu
- plik HTML z opisem stolicy
- katalog `img` z grafikami
- pliki dokumentacyjne (README, CONTRIBUTING)

## Wykorzystane technologie
- HTML5
- Git
- GitHub
- Trello (do zarządzania zadaniami)
